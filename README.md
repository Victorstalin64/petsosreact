## 📖 Descripción

**PetSOS** es una aplicación web desarrollada para facilitar la protección y el bienestar animal mediante herramientas digitales. La plataforma permite a los usuarios reportar casos de fauna urbana, gestionar procesos de adopción y acceder a información relevante con el apoyo de un asistente virtual basado en inteligencia artificial.

El proyecto fue desarrollado utilizando una arquitectura moderna basada en **React**, **Vite** y **Firebase**, priorizando el rendimiento, la escalabilidad y una experiencia de usuario intuitiva.

# ✨ Características

- 🐶 Plataforma para adopción de mascotas.
- 📍 Reporte de fauna urbana.
- 🤖 Chatbot inteligente (PetSOSBot).
- 🔐 Autenticación mediante Firebase.
- ☁️ Base de datos en Firebase Firestore.
- 📱 Diseño completamente responsivo.
- 📊 Generación de reportes en PDF.
- 📈 Visualización de estadísticas mediante gráficos.
- 🎨 Animaciones modernas con Framer Motion.
- 📞 Formulario de contacto.

# ⚙️ Instalación

## 1. Clonar el repositorio

```bash
git clone https://github.com/Victorstalin64/petsosreact.git
```

## 2. Entrar al proyecto

```bash
cd petsosreact
```

## 3. Instalar dependencias

```bash
npm install
```

## 4. Ejecutar el servidor

```bash
npm run dev
```

La aplicación estará disponible en:

```
http://localhost:5173
```

# 🔥 Firebase

El proyecto utiliza Firebase para:

Las credenciales deben configurarse mediante un archivo de configuración de Firebase antes de ejecutar el proyecto.
